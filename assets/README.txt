Coloca aquí las imágenes proporcionadas por el cliente con estos nombres:

- logo-kombi.png    -> logo principal (recomendado fondo transparente, aprox 512x512)
- facebook.png      -> icono de Facebook (ej. 128x128)
- instagram.png     -> icono de Instagram (ej. 128x128)
- whatsapp.png      -> icono de WhatsApp (ej. 128x128)
- maps.png          -> icono de ubicación/Google Maps (ej. 128x128)

Si no colocas las imágenes, la página usará los iconos de FontAwesome como fallback visual.
